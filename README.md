# EtherPay
## Ethereum Payment Processor

This is a proof of concept project demonstrating how to process ethereum payments with a Python 3.5 backend.

### Installation
run `npm install`

##### HTML client
run `npm run start`

##### Flask API
run `pipenv shell`
run `pip install src/backend/requirements.txt`
run `npm run server`

### Build
run `npm run build`
